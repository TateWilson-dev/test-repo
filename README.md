# test-repo

Test repository for validating the Pelycon secure vibe-coding workflow (branch -> PR -> Gitleaks -> GitHub Actions -> human review).

## About

This repo has no application code. It exists purely as a sandbox to exercise the Pelycon secure vibe-coding workflow end to end: creating feature branches, opening pull requests, running Gitleaks secret scanning, running GitHub Actions checks, and requiring human review before merge. Changes here are test changes, not production functionality.
